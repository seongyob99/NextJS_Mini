# Todo-List
Next.js와 Spring Boot를 이용한 미니 프로젝트 입니다.

## 기술 스택
### Front-end
- 프레임워크 및 라이브러리 : Next.js, Axios
- 언어 : JavaScript
### Back-end
- 프레임워크 및 라이브러리: SpringBoot, Spring Data JPA, QueryDSL
- DBMS : MariaDB
- 언어 : Java(17)
### Tools
- IDE : IntelliJ, VSCode
- Version Control : Git, Github
